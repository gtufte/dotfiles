# DotFiles

A collection of my dotfiles where:

* vim is the editor
* fish is the shell
* terminator is the terminal emulator

## PreReq

Install these packges before you run the script:

* vim
* fish
* terminator
* git
* curl

## Install / Usage

```bash
git clone https://github.com/gtufte/dotfiles.git
./dotfiles --help
```
